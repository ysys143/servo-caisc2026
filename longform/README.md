# Long-form originals

The pre-compression manuscripts from which the CAISc 2026 submission was cut.
Moved here from `../../ai_scientist/paper/` on 2026-07-20 so that this repository
is the single canonical location for the paper and its ancestors.

| file | what it is |
|---|---|
| `main_full.tex` | English long-form original (2026-05-31). 20 pages. Contains the per-system and per-domain analyses that the 8-page camera-ready compresses into two tables. |
| `main_ko_full.tex` | Korean long-form original, kept current through 2026-07-20. 34 pages. Superset of `../submission/main_ko.tex`: 26 subsections against 6, and 32 citations the compressed version does not carry. |
| `references_full.bib` | Bibliography for both, 97 entries. Kept separate from `../submission/references.bib` so that edits here cannot disturb the submitted bibliography. |
| `REVISION_NOTES.md` | Review-response notes, 2026-05-31. |

## Status

**Not the submission.** The archival version of record is
`../submission/servo_caiscfp2026.pdf`; the corrected post-submission manuscript is
`../submission/main_post-submit.tex`. Nothing here feeds the supplementary ZIP or the
public reproducibility repository.

These files are kept because the compression to eight pages dropped material that has
independent value — the full per-system decompositions, the seven-domain analyses, and
several open-problem discussions with citations that appear nowhere else. They are the
natural starting point for an extended version.

## Caveat

The long-form manuscripts have **not** been carried through the post-submission
correction pass applied to `../submission/main_post-submit.tex` (entries R1–R12 in its
Appendix D). The Korean long-form received part of that pass on 2026-07-20 — corrected
propositions, the experiment-fidelity open problem, the limitations paragraph, and the
citation-audit fixes that applied to it — before it was established that
`../submission/main_ko.tex` is the manuscript the citation audit tracks. The English
long-form has received none of it. Treat claims here as predating those corrections and
check against `main_post-submit.tex` before reusing any passage.

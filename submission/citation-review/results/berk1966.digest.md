## Thesis / Problem
Studies the LARGE-SAMPLE (k → ∞) behavior of a Bayesian posterior distribution P_k over a
parameter θ WITHOUT assuming the model is correct — i.e., the observations {X_i} are equally
and independently distributed (e.i.d.) with some true distribution F that need NOT equal any
density f(·|θ) in the parametric model. Question: where does the posterior concentrate when the
model is misspecified?

## Method
- Model: X_i e.i.d. with density f(·|θ), θ ∈ Θ (Borel subset of a complete separable metric space);
  prior P on Θ (Θ = carrier of P); posterior P_k from Eq. (1.1). Truth: X_i ~ F, arbitrary.
- Works with a possibly reweighted likelihood u(x|θ) = g(x)f(x|θ) (Eq. 2.1; leaves posterior
  unchanged), following Le Cam's device u = f(·|θ)/f(·|θ₀).
- Defines H(·|θ) = log u(·|θ); η(θ) = E_F H(X|θ) (expectation under the TRUE F); η* = sup_θ η(θ).
- Asymptotic carrier: **A₀ = {θ ∈ Θ : η(θ) = η*}**, the set of θ MAXIMIZING η (Eq. 2.3).
- Proof: adaptation of Le Cam ([7], pp.304–5) using a generalized SLLN for Banach-valued random
  variables (Hanš, Mourier), U-statistics, and martingale convergence (Doob).

## KL-MISSPECIFICATION RESULT — CONFIRMED
Eq. (2.2): **η(θ) ≤ E log f(X)** with equality **precisely when f(x|θ) is also a density for F**;
Berk states η(θ) "measures the similarity between F and f(·|θ)" and is "essentially a measure of
entropy," citing Kullback [6] (Information Theory and Statistics). Maximizing η(θ) = E_F[log f(X|θ)]
is exactly equivalent to MINIMIZING the Kullback–Leibler divergence D(F ‖ f_θ) (the E_F[log f_F]
term is θ-free). Hence **A₀ = the set of KL projections / KL-minimizers of the model onto the true F**,
generally distinct from any "true parameter" and possibly containing MORE THAN ONE point.
**Main Theorem (§4):** under Assumptions (i)–(iv), if X_i ~ F then P_k is almost surely [F]
asymptotically carried on A₀ — for every open U ⊇ A₀, lim_k P_k(U) = 1 [F]. This is the classical
Bayesian-under-misspecification concentration result.

## Scope
- i.i.d.-type (e.i.d.) observations only; single true distribution F.
- Establishes topological CONCENTRATION (asymptotic carrier), i.e., the support the posterior mass
  collapses toward — not a rate, not a Bernstein–von Mises limit.
- Θ Borel in complete separable metric space; Θ = prior carrier; requires the boundedness
  Assumptions (iii)–(iv) (dominated convergence / SLLN machinery).

## Does NOT claim / boundaries
- Does NOT claim the posterior converges to a limiting distribution OVER A₀; §5 counterexample shows
  it may fail to stabilize (oscillation) when A₀ has >1 point. (Contrast: correct-model case DOES
  converge to a point mass.)
- Does NOT give convergence rates or a Gaussian (BvM) shape.
- Does NOT require A₀ to be a single point or to contain any true parameter.
- Non-i.i.d. / dependent data, model-selection, and "non-degenerate weak limit" cases are left open
  (Berk notes hope to report the latter "in a subsequent paper").
